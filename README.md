# Fontmaker
Makes your own font.

**Dependencies**

* Opengl(GLUT, GL, GLU)
* OpenCV(CV2)

**Execution**

`$ python3 main.py`

and start typing.

*How you can use your own font*

Add your font images in the directory images/ and run the program.
